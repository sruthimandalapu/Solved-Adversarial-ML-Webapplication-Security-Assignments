Understanding Gradient Based Adversarial Attacks¶

Adversarial perturbations are small changes to an machine learning model's input with the goal of pushing that input over the model's decision boundary. Visualizing decision boundaries and understanding concepts from gradient-based adversarial attacks can be challenging in high dimensional spaces. This homework is adapted from the notebook https://github.com/adrian-botta/understanding_adversarial_examples/blob/master/adversarial_examples_logistic_regression.ipynb

We'll start with a simple classification example with a subset of the Iris dataset - (inspired from: https://medium.com/@martinpella/logistic-regression-from-scratch-in-python-124c5636b8ac)

When we train our machine learning models, we're defining a loss function that we then aim to minimize $$J(\theta) = \frac{1}{m}\sum_{i = 1}^M \log(1+\exp(-y_i \theta^Tx_i)),$$ where $y_i$ is the label of the ith sample, and $x_i$ is a two-dimensional vector representing the feature of the ith sample.

def gradient_wrt_theta(theta, X, y):
    """
    Compute the gradient of logistic loss with respect to theta.

    Args:
        theta: (1,2) numpy array, parameter vector
        X: (m, 2) numpy array, input features
        y: (m,) numpy array, labels

    Returns:
        grad: (1,2) numpy array, gradient vector
    """
    z = np.dot(X, theta.T).flatten()
    exp_term = np.exp(y * z)
    grad = -np.dot(X.T, y / (1 + exp_term)) / y.shape[0]
    return grad.reshape(1, -1)

With gradient-based adversarial attacks, we want to use the gradients to determine how to change X such that the loss increases (or decreases with a targeted attack). Implement the fast gradient sign method-based adversarial sample generation, i.e., $$X' = X + \eta = X + \epsilon\times \mathrm{Sign}\left( \frac{\partial J(\theta)}{\partial X} \right)$$ (please evaluate)

import numpy as np

def sign_gradient_wrt_X(theta, y):
    """
    Compute the gradient of the loss with respect to input features X,
    optionally returning signed (normalized) gradients.

    Args:
        theta: (1, n_features) numpy array, model parameters
        X: (m, n_features) numpy array, input features
        y: (m,) numpy array, labels in {0, 1} or {-1, +1}
        normalize: bool, whether to return unit-norm gradients

    Returns:
        grad: (m, n_features) numpy array, gradients for each sample
    """
    num_samples = X.shape[0]
    p=y*(X @ theta.T).flatten()
    gradient=-(y[:, None]*theta)/(1 + np.exp(p))[:, None]
    grad_sign = np.sign(gradient)
    return grad_sign
